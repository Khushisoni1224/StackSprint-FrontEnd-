üü¶ SQL Basics (MySQL / PostgreSQL)

Structured Query Language (SQL) is the most widely used language for managing relational databases. It allows you to define schemas, insert, update, and query data using declarative statements. SQL is fundamental for any backend or data-driven application.

CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name VARCHAR(100),
  Age INT
);

SELECT * FROM Students;

üü® PL/SQL (Oracle)

PL/SQL (Procedural Language/SQL) is Oracle's extension of SQL that supports procedural constructs like loops, conditions, and exception handling. It's used to write complex programs like stored procedures, functions, and triggers within Oracle databases.

DECLARE
  v_name VARCHAR2(50);
BEGIN
  SELECT name INTO v_name FROM Employees WHERE ID = 101;
  DBMS_OUTPUT.PUT_LINE('Name: ' || v_name);
END;

üü© MongoDB (NoSQL)

MongoDB is a NoSQL database that stores data in flexible, JSON-like documents. It does not require a predefined schema, making it ideal for unstructured and semi-structured data. MongoDB is widely used in modern web applications due to its scalability.

db.students.insertOne({
  name: "Amit",
  course: "CS",
  marks: 90
});

db.students.find({ marks: { $gt: 80 } });

üü™ Neo4j (Graph DB)

Neo4j is a graph database that stores data as nodes, relationships, and properties. It uses the Cypher query language to express graph patterns, and is ideal for social networks, fraud detection, and recommendation systems where relationships matter most.

CREATE (a:Person {name: 'Alice'})
CREATE (b:Person {name: 'Bob'})
CREATE (a)-[:FRIENDS_WITH]->(b);
